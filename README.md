This tool is still in development and is subject to change.

This tool parses Roman visit files (WFI Fundamental Observational Sequence template only) and plots the results. Examples of visit files are not provided at this point in time until they are confirmed cleared.

<img width="2000" height="900" alt="V0010701001001013001_03" src="https://github.com/user-attachments/assets/93350ce1-ee6e-47d1-a058-03aad9bc26f9" />

Installation:
- git clone https://github.com/Skyhawk172/roman_visit_viewer.git
- pip install -e .

Command-line execution:
- roman_visit_viewer /Path/to/visit/files

Note that you can specify either a .vst file or a directory containing .vst files, and the script will process them all at once.
